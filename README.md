# hw-03-neuromafia

Это задание без готового CI или заготовки кода. 

## Описание

Нужно реализовать симулятор игры в мафию. Для реализации ботов предлагается использовать LLM-ки.

В качестве UI используется текстовый CLI. В базовой версии боты играют между собой.
В качестве бонуса — реализация режима игры с участием человека.

### Правила игры в мафию

Полные правила можно смотреть здесь: https://dom-mafia.ru/sport_mafia_game_rules.

Каждый игрок в партии либо мафия, либо мирный житель. Всего 10 игроков, 3 мафии, 1 из которых дон, 7 мирных жителей, один из которых шериф.

Победа команды мирных наступает в тот момент, когда из игры выведены все представители мафии.
Победа мафии наступает в случае, когда за игровым столом остаётся равное количество игроков обеих команд или количество мафиози больше, чем мирных.
Моментом вывода игрока является момент голосования или ночного убийства.

Игра состоит из циклов:
- День
  - Обсуждение
  - Голосование
- Ночь
  - Убийства
  - Проверка шерифа

#### День. Обсуждение

Во время обсуждения игроки по очереди по часовой стрелке высказывают свое мнение относительно того, кто является мафией:
- начинают с первого игрока
- в каждый следующий день очередность сдвигается на один по часовой стрелке
Также игроки во время своего хода могут выставить одного игрока на голосование, среди тех, кто еще не был выставлен.

#### День. Голосование

Во время голосования игроки могут проголосовать за выставленных игроков.
Ведущий объявляет игроков в том порядке, в каком они были выставлены на голосование, все игроки либо голосуют, либо пропускают голосование.
Каждый игрок может проголосовать не более, чем за одного игрока.
Игрок, за которого проголосовали больше, чем за других, покидает игру.
При наличии нескольких игроков с одинаковым количеством голосов проводится повторное голосование.
Перед повторным голосованием подозреваемые игроки имеют право на дополнительное высказывание.
Если голосов вновь равное количество, проводится голосование "выгнать ли всех игроков, набравших максимум голосов".

Роль покинувшего игрока не раскрывается.

#### Ночь. Убийства

Ночью мафия выбирает одного живого игрока, которого хочет вывести из игры.
Мафия не может выбрать игрока, который уже покинул игру.
Если среди мафии осталось несколько игроков, решение принимается большинством голосов мафии; при равенстве голосов выбирается кандидат дона.
Если дон уже покинул игру или не выбрал одного из лидирующих кандидатов, кандидат выбирается случайно среди игроков, набравших максимум голосов мафии.
Выбранный игрок покидает игру в конце ночи.

Роль убитого игрока не раскрывается. Убитый игрок имеет право на последнее слово.

#### Ночь. Проверка шерифа

Если шериф жив, ночью он может проверить одного живого игрока.
Шериф не может проверить самого себя и игроков, которые уже покинули игру.
В результате проверки шериф узнаёт только команду выбранного игрока: мафия он или мирный житель.
Конкретная роль игрока не раскрывается: например, дон определяется как мафия, а обычный мирный и шериф — как мирные жители.

Если шериф покинул игру, проверка ночью не проводится.
Результат проверки известен только шерифу и не объявляется всем игрокам автоматически.
